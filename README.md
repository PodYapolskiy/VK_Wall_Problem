# VK Wall Problem

The goal of this code is to delete heap of reposted posts on VK wall, which are not needed.

## USAGE

1. Copy code and create `venv` with given requirements.
2. Create file `config.py` and put 2 string variables: `login`; `password`.
   You should use only your phone number as login.

   ```python
   login = '+7**********'
   password = '*********'
   ```
3. Run code with following command:

   ```shell
   python main.py
   ```
